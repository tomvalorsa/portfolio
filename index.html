Stuff I like to use:

Online Football Manager

An online sports management game built predominantly in Backbone.js with a Ruby on Rails back-end. Users join a league and choose a team to manage, fine-tuning tactics and competing against other users on a daily basis. This is a prototype version that I used to model a current pet project which I continue to add to in my spare time. Technologies used include: Backbone.js, Underscore.js, jQuery, Ruby on Rails, Bootstrap, Postgres, RSpec and Chart.js.

Glucosy

Proof of concept utility tool for sufferers of type 1 diabetes. This app allows users to visualise information about their bloodsugars. They can also make use of a meal calculator to see how much insulin is needed to offset the amount of carbohydrates in a meal. Users will also be notified of harsh temperature changes in the near future via email (as this can have a significant effect on bloodsugar levels). Technologies used include: Dropzone.js, Chart.js, Ruby on Rails, jQuery, Bootstrap, Underscore.js, Geocoder and Mandrill.

Snip{it}

A site built to foster front-end creativity with inspiration galleries and code snippets for use in web-based projects. Users can post and favourite various images and code snippets which relate to website components (e.g. buttons and nav bars). Technologies used include: Ruby on Rails, Bootstrap, AWS Image Upload, Carrierwave, Animate.css and Postgres.
